## 关于这个博客

这个博客使用 Jekyll + GitHub Pages 搭建，写作流程非常简单：

1. 在 `_posts/` 目录下新建一个 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-文章标题.md`（例如 `2026-03-11-hello-world.md`）
3. 在文件开头加上 front matter（就是两行 `---` 之间的内容）
4. 写完 push 到 GitHub，稍等片刻就能在网页上看到了

## 文章模板

每篇文章开头需要这样的格式：

```
---
title: "文章标题"
date: 2026-03-11
---

正文内容从这里开始...
```

就这么简单！
